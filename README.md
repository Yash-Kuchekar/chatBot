# ChatBot Using AI #

This is a Python-based chatbot project that uses LangChain, Streamlit, and PyPDF2 for document-based question answering.

# Features
- Uses AI to answer questions from PDFs.
- Built with Python and Streamlit.
- Simple and user-friendly interface.

# How to Run
1. Install required libraries:
   pip install streamlit langchain PyPDF2
2. Run the app:
   streamlit run chatBot.py
